## 📋 Sobre o Projeto

O **Sistema Hospital** é uma aplicação desktop desenvolvida em Java que centraliza o gerenciamento de informações hospitalares. O sistema permite registrar e consultar dados de pacientes, médicos e consultas de forma organizada.

---

## ✨ Funcionalidades

- 👤 **Cadastro de Pacientes** — Registro de dados pessoais e histórico dos pacientes
- 👨‍⚕️ **Controle de Médicos/Funcionários** — Gerenciamento da equipe médica
- 📅 **Registro de Consultas** — Controle e acompanhamento de consultas realizadas

---

## 🛠️ Tecnologias Utilizadas

- **Java** — Linguagem principal
- **Maven** — Gerenciamento de dependências e build
- **IntelliJ IDEA** — IDE de desenvolvimento

## 🚀 Como Rodar Localmente

### Pré-requisitos

- [Java 17+](https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html)
- [Maven 3.8+](https://maven.apache.org/download.cgi)

### Passos

```bash
# 1. Clone o repositório
git clone https://github.com/DioguinhoGamer3D/sistema-hospital.git

# 2. Entre na pasta do projeto
cd sistema-hospital

# 3. Compile o projeto
mvn clean install

# 4. Execute a aplicação
mvn exec:java
```
